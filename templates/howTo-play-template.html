Create game

To create a game you need to go to create page. You can see there are the rules in left. fill the form with typing the name and choosing your symbol. Also you can make the game private with password protection. After that you can share the game with friends.

Join game

Go to all games page and search for orange button with text "Join". Click it and play

In game

When you are in the game, first check the status. It will tell you what action to do. If there are no another opponent in the game you should wait. If it's your turn - click on the empty tile you want to place mark and then click play button else should wait until opponent plays. Pop up messages are showing from top-right of the screen if you make mistake.

- All operations needs a registered account.
